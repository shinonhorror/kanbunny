# Kunbunny Backend

Это backend для проекта Kunbunny. Сейчас проект небольшой, поэтому самое удобное место для старта - папка [`docs`](./docs).

Если ты впервые пишешь backend, начни отсюда:

1. [`docs/start-here.md`](./docs/start-here.md) - как подготовить `.env`, поднять базу и запустить приложение.
2. [`docs/project-structure.md`](./docs/project-structure.md) - что где лежит и как файлы связаны между собой.
3. [`docs/commands.md`](./docs/commands.md) - команды проекта простыми словами.
4. [`docs/backend-basics.md`](./docs/backend-basics.md) - мини-словарь backend, NestJS, Prisma и PostgreSQL.
5. [`docs/api.md`](./docs/api.md) - текущие API-ручки и Swagger.

Коротко о стеке:

- NestJS - backend-фреймворк на TypeScript.
- Prisma - работа с базой данных.
- PostgreSQL - база данных.
- Docker Compose - локальный запуск PostgreSQL.
- Swagger - страница с документацией API.

Быстрый запуск описан в [`docs/start-here.md`](./docs/start-here.md).
